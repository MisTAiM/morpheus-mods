📋 Install Guide

How to Install — Step by Step

Follow this order exactly. Skipping steps causes crashes.

1
Download BG3 Mod Manager
Download and extract BG3ModManager to a non-protected folder (not Program Files). Run BG3ModManager.exe — it will auto-detect your game path. Run the game once first so folders get created.
↗ LaughingLeader / GitHub
2
Install Script Extender via BG3MM
In BG3MM, go to Tools → Download & Install Script Extender. DO NOT download SE from Nexus — that's an unofficial reupload. Only use Norbyte's GitHub. After install, launch the game once so SE fully installs. Look for "Script Extender version XX loaded" in the main menu bottom-left.
↗ Norbyte / GitHub
3
Download Mods from Nexus / GitHub
Go to each mod page. Click Manual Download → Slow Download (free). Save the .zip or .pak file. Do NOT use Vortex Mod Manager — BG3MM is far more reliable for BG3.
4
Drag .pak Files into BG3MM
Drag downloaded zip files directly into the BG3MM window. A green bar at the bottom confirms successful upload. Move mods from "Inactive" to "Active" column (leave DiceSet_06 and Honour mods out).
5
Set Load Order — Foundation First
In active mods, order matters. Load order: (1) Mod Fixer, (2) Native Mod Loader, (3) ImprovedUI, (4) Script Extender mods, (5) Content/gameplay mods, (6) Visual mods last.
6
Export to Game and Launch
Click "Export Order to Game" (first export button) or File → Export Order to Game. Then launch BG3 from BG3MM. Verify Script Extender shows in the menu. New save recommended for mods marked New Save Required.
